DMeXa

Date: Sept 2019
Download: ESP8266 Arduino Image
Source Code: Github
Whenever I walked into our school's auditorium, I would jokeingly say "Alexa, Turn on the hosue lights" of course, it would never work but I would always give myself a chuckle. This code, nicknamed "D-M-X-a," a combination of "DMX" and "Alexa." As in the name, it uses DMX to control specific fixtures in a DMX controlled environment. It uses an NodeMCU 1.0 ESP8266 and a Max485 Serial converter board. It acts as a DMX master device, and allows control over 1 DMX universe.
